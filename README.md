# Software yang dibutuhkan
- Xampp
- Visual Studio Code
- Node JS
- Web Browser (Chrome)

# Running Application
- import db coffe from dir arkademy/6/6c/db_dump/caffe.sql
- open terminal dir arkademy/6/6c
- type node app.js & press enter
- open web browser, type localhost:8000 & press enter
